Tecnologías Utilizadas

  • Django: Framework web de alto nivel para Python que permite el desarrollo rápido y limpio de aplicaciones web. En este proyecto, Django se utiliza para gestionar la lógica del servidor, la autenticación de usuarios y la interacción con la base de datos.
  • GAMA Headless: Plataforma de simulación avanzada que permite la creación y ejecución de modelos complejos sin una interfaz gráfica, ideal para entornos de servidor.
  • PostgreSQL: Sistema de gestión de bases de datos relacional de código abierto, conocido por su robustez y rendimiento. Almacena todos los datos relacionados con los usuarios, las simulaciones y los resultados de las simulaciones.
  • Nginx: Servidor web y proxy inverso de alto rendimiento. Se utiliza para servir la aplicación web, manejar las solicitudes HTTP y actuar como un proxy inverso para Gunicorn, el servidor de aplicaciones de Django.
  • Docker: Plataforma que permite la creación y gestión de contenedores, proporcionando un entorno aislado y reproducible para las aplicaciones. El archivo docker-compose.yml define los servicios necesarios y sus configuraciones.
  • Vue.js (Futuro): Framework progresivo para construir interfaces de usuario. En futuras actualizaciones, Vue.js se integrará en la plataforma para mejorar la experiencia del usuario, proporcionando una interfaz más dinámica y reactiva para gestionar y visualizar las simulaciones.

Control de Usuarios y Permisos

La plataforma utiliza el sistema de autenticación y permisos de Django para gestionar el acceso de los usuarios. Los usuarios pueden registrarse, iniciar sesión y cerrar sesión. Además, se pueden asignar diferentes roles y permisos a los usuarios para controlar el acceso a las distintas funcionalidades de la plataforma.

Gestión de Simulaciones

GAMA Headless se utiliza para ejecutar las simulaciones definidas por los adminisradores. Los resultados de las simulaciones se almacenan en la base de datos PostgreSQL y pueden ser analizados y visualizados a través de la plataforma web. Esto permite a los usuarios ejecutar simulaciones complejas directamente desde su navegador web, sin necesidad de instalar software adicional.
